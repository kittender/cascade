Cascade

A modern and flexible CSS framework.

Layout

Box, stack, center, cluster, switcher concepts from Every Layout.

Box

To create a box context for content, use the .box class on the container.

The padding can be adjusted with .is-size classes.

Media

To create a media presentation for content, use the .media class on the container.

The gap can be adjusted with .is-size classes.

Stack

To get automatic vertical rythm, use the .stack class on the container.

The vertical spacing can be adjusted with .is-size classes.

Center

To create a center element, use the .center class on the element.

The max-width it can take is defined by the setting $measure.

Cluster

To get automatic horizontal rythm, use the .cluster class on the container, and add an intermediate container, like a simple div, to absorb horizontal margins.

The horizontal spacing between clustered elements can be adjusted with .is-size classes.

Switcher

To get automatic fluid columns, use the .switcher class on the container, and add an intermediate container, like a simple div, to absorb horizontal margins.

Define a max quantity of items beyond which it switches to vertical stack with .has-maxItems classes, from 2 to 12.

The vertical spacing can be adjusted with .is-size classes.

The threshold under which it always turns into stacked elements, is defined by the setting $measure.

Grid

To create grid, use the .grid class on the container, and add an intermediate container, like a simple div, to absorb horizontal margins.

The quantity of columns can be set with .is-quantity classes, from 2 to 12.

The gap can be adjusted with .is-size classes.
